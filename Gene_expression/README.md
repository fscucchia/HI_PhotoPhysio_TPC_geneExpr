## Gene expression analysis

- scripts for [cleaning, aligning and assembling](https://github.com/fscucchia/HI_PhotoPhysio_TPC_geneExpr/tree/main/Gene_expression/scripts/Filtering_Mapping) raw RNA sequences, for [functional annotation](https://github.com/fscucchia/HI_PhotoPhysio_TPC_geneExpr/tree/main/Gene_expression/scripts/Functional_Annotation), for [WGCNA and GO enrichment](https://github.com/fscucchia/HI_PhotoPhysio_TPC_geneExpr/tree/main/Gene_expression/scripts/WGCNA_Network_Analysis), for [gene expression plasticity](https://github.com/fscucchia/HI_PhotoPhysio_TPC_geneExpr/tree/main/Gene_expression/scripts/Gene_Expression_Plasticity)
- input data for all analyse in the [data directory](https://github.com/fscucchia/HI_PhotoPhysio_TPC_geneExpr/tree/main/Gene_expression/data).
